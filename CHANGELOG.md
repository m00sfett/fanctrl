# Changelog

All notable changes to this repository are documented in this file.

## 1.0.0 - 2026-03-03
- Initial stable release of `fanctrl`.
- Temperature-based fan control with configurable ON/OFF thresholds and hysteresis.
- GPIO control via `libgpiod` (modern GPIO character device interface).
- Built-in status endpoint for integration with containerized monitoring/overview services.
